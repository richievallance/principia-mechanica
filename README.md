## Abstract

*Principia Omnium Universale* establishes Leonardo da Vinci's unified philosophical and mechanical architecture as documented in Madrid Codex II (BNE MS 8936). This is a Da Vinci document, not a Universal Simulation Construct (USC) paper. It serves as the bridge between the mechanical treatise of Madrid Codex I (*Da Valenca Mechanica*) and the broader argument that Leonardo's four-element framework constitutes a coherent physics of reality. Primary evidence is drawn from BNE folio citations throughout.
